\section*{Abstract}
Fitting many figures into very tight manuscripts while keeping it pleasant to read is challenging. This \LaTeX template shows how to generate a compiled PDF with figures embedded into the text. Throughout the body of the text you will find different examples of how to embed figures or tables. This is supposed to give you a range of options from which you should choose the one best suited for your data. Check figure legends and text for more detailed explanations. Check out \cite{Schlegel2016} as example of use.

\section*{Introduction}
In the introduction you will see an example of text wrapping around the figure with a figure legend at the margin. This is done by combining the \verb!wrapfigure! with the \verb!marginnote! environment. Please note that in this case the figure (\verb!wrapfigure!) and %avoid blank space here 
\marginpar{\vspace{.7cm} \color{Gray} % adjust vertical position relative to text with \vspace{}
\textbf{Figure 1. Example of a margin caption.}
\textbf{A}, Setting up your figure and legend like this looks fancy but requires a bit more work because you have to take care of positioning and figure numbers yourself. Important to note is that you can't use \textbackslash label and \textbackslash ref for margin captions - you will need to do this manually. 
}
% the number in [] of wrapfigure is optional and gives the number of text lines that should be wrapped around the text. Adjust according to your figures height
\begin{wrapfigure}[18]{l}{83mm}
\includegraphics[width=83mm]{fig1.pdf}
\end{wrapfigure} % avoid blank space here
the figure legend (\verb!marginpar!) have to be separated as you can tell from the code. The \verb!wrapfigure! environment can be a bit tricky when it comes to text formatting. Thus some general hints: (a) try to avoid line breaks in the code as this may result in weird formatting around the figure, (b) the figure should not span across multiple headlines (sections) and (c) if you encounter problems with the line break right after the \verb!wrapfigure! try using \verb!\mbox{}! to prevent premature line breaks (\mbox{see example in code}). As stated in the figure legend (Fig. 1), the \verb!\label{}! and \verb!\ref{}! command won't work with this way to setup your figures. The reason is that \verb!\label{}! only works if there is a caption in the figure environment which we don't have in this case because we manually moved the caption to the margin.
